# View specific Authorisation Code linked to an account `(viewServiceAvailability)`

Here, `viewServiceAvailability` creates a request to /heartbeat

## Usage/Examples

### Example Input

```
{
  "type": "viewServiceAvailability"
}
```

### Example Output

onSuccess:

```
response:
{
  "serviceStatus": "available"
}

status:
    200
```
